HyperText Markup Language is a language that gives a structure and meaning to the content it describes.

HTML tags

paragraph tag
used for adding a text paragraph
image tag
is used to add an image on the page
quotation tag
is used to add a small quotation embeded within other content
section tag
group of related content
anchor tag
links pages together
